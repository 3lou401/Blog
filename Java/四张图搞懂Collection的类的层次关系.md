# Collection vs Collections

首先，“Collection”和“Collections”是两个不同的概念。从下面的层次关系图也可以看到。“Collection”是Collection层次关系中的一个接口，但是“Collections”是一个类，这个类提供了可以用于某些Collection类型的静态方法！
Arrays也是一个类似的类，提供一些静态方法。
